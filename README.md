# BirdBox Dashboard

A web-based dashboard for visualizing bird box usage data collected by the Genesee Land Trust.

This project focuses on data processing, storage, and visualization—transforming sensor data into meaningful insights about bird activity, species presence, and box utilization.

---

## Overview

The BirdBox Dashboard enables researchers and staff to:

- Track bird box usage over time
- Identify species occupying each box
- Monitor vacancy vs occupancy rates
- View historical and real-time data through an intuitive interface

Note: This repository does not include hardware or sensor code. It assumes data is already collected and transmitted to the system.

---

## Features

### Backend
- REST API for ingesting and retrieving bird box data
- Secure data handling and storage
- Timestamp-based event tracking
- Species classification storage (from external systems)

### Frontend
- Interactive dashboard for data visualization
- Temporal data views (daily, weekly, seasonal trends)
- Bird box status indicators (occupied / vacant)
- Simple UI designed for low-training users
